Sistema simples em Python para gerenciamento de gastos utilizando arquivo CSV como base de dados.

Esse projeto foi desenvolvido com o objetivo de praticar:
- Manipulação de arquivos
- Leitura e escrita em CSV
- Estruturas de repetição
- Estruturas condicionais
- Organização de código com funções

Entre as funcionalidades estão:
- Adicionar gasto
- Listar todos os gastos
- Calcular total da fatura
- Excluir um gasto específico
- Criação automática do arquivo "gastos.csv"

Foram usados para manipulação de arquivos os modos:
- "w" para escrita
- "a" para adicionar
- "r" para leitura

Biblioteca CSV para estruturar dados em formato de tabela.
